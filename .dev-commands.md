# Quick Reference: Development Commands

## Setup

```bash
source venv/bin/activate
```

## Common Commands

### Format Code
```bash
venv/bin/black *.py
```

### Lint Code
```bash
venv/bin/ruff check *.py --fix
```

### Run Tests
```bash
venv/bin/pytest tests/ -v
```

### Run All Checks
```bash
venv/bin/black *.py && venv/bin/ruff check *.py && venv/bin/pytest tests/ -v
```

### Download NLTK Data
```bash
venv/bin/python scripts/download_nltk_data.py
```

### Interactive Shell
```bash
venv/bin/ipython
```

## Test Specific Things

```bash
# Run one test file
venv/bin/pytest test_pos_tagger.py -v

# Run one test class
venv/bin/pytest tests/test_sample_corpus.py::TestSampleCorpusFrequencies -v

# Run one specific test
venv/bin/pytest tests/test_sample_corpus.py::TestSampleCorpusFrequencies::test_sample1_token_count -v
```

## Quick Analysis

```bash
# Analyze sample corpus
venv/bin/python -c "import pos_tagger as pt; print(pt.run('data/sample/sample1_dialectics.txt')['nouns'][:10])"

# Search for term
venv/bin/python -c "import pos_tagger as pt; print(len(pt.search_term_in_file('philosopher_1920_cc.txt', 'consciousness')))"
```
